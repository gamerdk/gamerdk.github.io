DevFile10 (Calculator)
  • naamProject: Calculator
  • programmeertaal: C# (favoriete programmeertaal)
  • Solo Project voor school
  • Tema:
    • UI
    • Calculator
  • devices: Windows,Mac,Linux

Deze DevFile is de eerste project waar ik werk met Unity. Iedereen kreeg de UI en de Script met basis functies die zowiso gebruikt werden. Met dat ging iedereen eigen code schrijven om de Calculator werkend te maken. Ik heb uiteindelijk het zo uitgebreid gemaakt dat ik de reken volgorde heb toegepast, met commagetallen kan werken, met negatiefe getallen werken, modulo toe kan passen en het laaste andwoord weer hergebruiken zo vaal als je maar wilt. Dit terwijl er nog geen manier is gevonden om mijn calculator te breken. Het mooiste aan dit project is dat ik later een deel van de code in een ander project heb toegepast en de reacties van mijn groepje over de code was geweldig.
