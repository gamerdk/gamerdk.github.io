ISV B.V.

ISV is opgericht in 2001. Met het product ISV Software. ISV B.V. is het opgericht door één van de broers Brünen. Je herkent Brünen waarschijnlijk door de rijschool Brünen, maar ISV is opgericht door de andere broer. ISV B.V. is in 2021 overgenemen met alle producten en personeel die erbij kwamen. Tegenwoordig noemen wij ISV onder de handelsnaam Autofox.

ISV Software

ISV Software is het eerste product van ISV Software. Uiteindelijk werd ISV Software vervangen voor autofox wegens veroudering van Software.

Autofox

Autofox was een concurrent aangezien het een zeer flexibele Sofware is voor Rijscholen. Verder is het ook een ideaal opleidingsplatform voor andere bedrijfen die bijvoorbeeld chauffeur hebben die code 95 moeten bijhouden.
