Producten

Bij Yeshugo kan je Software aanschaffen via pakketten. Deze pakketten betaal je per voertuig per maand exclusief de Ecoboxen. Daarnaast kan je de voertuigen koppelen met YesHugo waardoor de Ecoboxen niet meer perse nodig zijn. Dit kan bijvoorbeeld met Tesla, Audi, Volkswagen en Skoda.

YesHugo

In 2021 is YesHugo ontstaan uit GreenStar. YesHugo heeft veel producten overgenomen van GreenStar, maar heeft de focus vooral op rittenregistratie voor bedrijven met (lease)auto's. Verder wil YesHugo de koppeling die mogelijk is met de 4 auto merken uitbreiden naar meer merken om op die manier minder afhankelijk te zijn van de ecoboxen. Ook willen ze meer gaan doen met hun API waardoor ook andere bedrijven zoals Dation gegevens per auto op kan halen uit YesHugo.
